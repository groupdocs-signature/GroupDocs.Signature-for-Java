# GroupDocs.Signature for Java

> A Java library to create, apply, verify, and remove digital and electronic signatures from documents including PDFs, Word, Excel, Images, and more. Supports text, image, barcode, QR code, stamp, metadata, and certificate-based digital signatures.

## About

This repository contains examples and demos for GroupDocs.Signature for Java, a powerful and flexible digital signature API for Java applications. It supports more than 90 document formats and enables developers to build robust signature workflows in enterprise systems.

**Key Features:**
- Digital signatures using X.509 certificates (PKCS#12, PKCS#7)
- Text, image, barcode, and QR code signature support
- Add stamp, metadata, and form field signatures
- Signature verification and removal capabilities
- Cross-platform compatibility (Windows, Linux, macOS)
